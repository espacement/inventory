# 📋 Учёт имущества — Flask-приложение

Простое мобильное веб-приложение для инвентаризации.

## Структура файлов

```
inventory/
├── app.py                  ← главный файл приложения
├── requirements.txt        ← зависимости Python
├── inventory.db            ← база данных (создаётся автоматически)
├── static/
│   └── uploads/            ← загруженные фото (создаётся автоматически)
└── templates/
    ├── base.html           ← базовый шаблон (шапка, навигация)
    ├── index.html          ← список предметов
    └── form.html           ← форма добавления / редактирования
```

---

## 🚀 Установка и запуск

### 1. Установи Python (если нет)
Скачай с https://python.org — версия 3.10 или новее.

### 2. Установи зависимости
Открой терминал (командную строку) в папке `inventory/`:

```bash
pip install -r requirements.txt
```

### 3. Запусти приложение

```bash
python app.py
```

### 4. Открой в браузере

- На этом же компьютере: http://127.0.0.1:5000
- С телефона в той же сети Wi-Fi: http://<IP-компьютера>:5000
  (IP можно узнать командой `ipconfig` на Windows или `ifconfig` на Mac/Linux)

---

## 📱 Использование с телефона

1. Компьютер и телефон должны быть в одной Wi-Fi сети
2. Узнай IP компьютера (например `192.168.1.100`)
3. На телефоне открой `http://192.168.1.100:5000`
4. Можно добавить в «Домой» как приложение через меню браузера

---

## ✨ Функции

- ➕ Добавление предметов (инв. номер, название, место, статус, комментарий)
- 📷 Фото предмета (съёмка камерой телефона прямо из браузера)
- 🔍 Поиск по любому полю
- 🏷 Фильтрация по статусу (Активен / Сломан / В ремонте / На списание / Зарезервирован)
- ✏️ Редактирование и удаление
- 📊 Экспорт в Excel (с подсветкой по статусу)
- 📈 Мини-статистика на главной

---

## 💡 Советы

**Как добавить предмет с телефона:**
1. Нажми «➕ Добавить» внизу
2. Введи инвентарный номер (можно с бумажки)
3. Нажми «📷 Сделать фото» — откроется камера
4. Выбери статус
5. Нажми «Добавить»

**Экспорт в Excel:**
Нажми «📊 Excel» внизу — скачается файл с текущим фильтром и поиском.

## 🔧 Продакшн (если нужно запустить на сервере)

Замени в `app.py` последние строки на:
```python
if __name__ == "__main__":
    os.makedirs(UPLOAD_FOLDER, exist_ok=True)
    init_db()
    app.run(debug=False, host="0.0.0.0", port=5000)
```

Для полноценного сервера используй `gunicorn`:
```bash
pip install gunicorn
gunicorn -w 2 -b 0.0.0.0:5000 app:app
```
